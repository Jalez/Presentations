What is JavaScript?

  • High-level, interpreted programming language
  • Used for both client-side and server-side scripting
  • Allows for interactivity and dynamic content on websites

Understanding the DOM Structure

The Document Object Model (DOM) is a tree-like representation of an HTML document. JavaScript can be used to interact with and manipulate the DOM.

  • Root node: The document itself
  • Element nodes: HTML elements, such as body, header, and div
  • Attribute nodes: Attributes, such as class and id, of HTML elements
  • Text nodes: Text content within HTML elements

How JavaScript Interacts with the DOM Tree Elements

JavaScript can be used to select, access, and modify elements in the DOM tree. This allows for dynamic manipulation of web pages and user interactions.

  • Selecting elements: getElementById, getElementsByClassName, querySelector
  • Accessing and modifying attributes: getAttribute, setAttribute
  • Changing element content: innerHTML, innerText, textContent
  • Adding and removing elements: createElement, appendChild, removeChild

Selecting and Accessing Elements with JavaScript

Selecting Elements

JavaScript provides various methods to select elements from the DOM:

  • getElementById: Selects a single element by its unique ID
  • getElementsByClassName: Selects all elements with a specific class name
  • getElementsByTagName: Selects all elements with a specific tag name
  • querySelector: Selects the first element that matches a CSS selector
  • querySelectorAll: Selects all elements that match a CSS selector

Accessing Element Attributes

You can access and modify an element's attributes using JavaScript:

  • getAttribute: Retrieves the value of a specified attribute
  • setAttribute: Sets the value of a specified attribute
  • removeAttribute: Removes a specified attribute
  • hasAttribute: Checks if an element has a specified attribute

Accessing Element Content

JavaScript can be used to access and modify the content of an element:

  • innerHTML: Gets or sets the HTML content of an element
  • innerText: Gets or sets the visible text content of an element
  • textContent: Gets or sets the text content of an element, including hidden text

Accessing Element Styles

JavaScript can be used to manipulate the styles of an element:

  • style: Accesses and modifies the inline styles of an element
  • getComputedStyle: Retrieves the computed styles of an element

Working with Element Content using JavaScript

Creating and Inserting Elements (using methods)

You can create and insert new elements into the DOM using JavaScript:

  • createElement : Creates a new element
  • append : Inserts a new element as the last child of a parent element
  • prepend : Inserts a new element as the first child of a parent element

Other methods for creating and inserting elements: appendChild, createTextNode , insertBefore createDocumentFragment , cloneNode , insertAdjacentHTML ,

Modifying Element Content (using attributes)

You can use JavaScript to modify the content of an element:

  • innerText: Gets or sets the visible text content of an element
  • textContent: Gets or sets the text content of an element, including hidden text
Not adviced to use:
  • innerHTML : Gets or sets the HTML content of an element
  • outerHTML : Gets or sets the HTML content of an element, including the element itself

Removing and Replacing Elements

JavaScript allows you to remove and replace elements in the DOM:

  • remove : Removes an element from the DOM
  • removeChild : Removes a specified child element from a parent element
  • replaceChild : Replaces a specified child element with a new element

Cloning Elements

JavaScript can be used to create a copy of an existing element:

  • cloneNode: Creates a copy of an element, with the option to include or exclude child elements

Event Handling in JavaScript

Introduction to Events

Events are actions or occurrences that happen in the browser:

  • Clicks
  • Mouse movements
  • Keypresses
  • Form submissions
  • And many more...

JavaScript can detect and respond to these events.

Learn more on MDN

Event Handlers and Listeners

Event listeners are functions that are called when an event occurs:

  • addEventListener : Attaches an event listener to an element
  • removeEventListener: Removes an event listener from an element

You can have multiple listeners for the same event on the same element.

Learn more on MDN

Event Types and Objects

When an event is triggered, an event object is created:

  • Contains information about the event
  • Can be used to control the event's behavior

Some useful properties:

  • type: The type of event
  • target: The element that triggered the event
  • currentTarget: The element that the event listener is attached to
  • preventDefault: Prevents the default action of the event
  • stopPropagation: Stops the event from up or down the DOM (Advanced stuff)

Learn more on MDN

Advanced Event Handling Techniques with JavaScript

Passing Parameters to Event Handlers

Use an anonymous function to pass parameters to event handlers:


        element.addEventListener('click', function(event) {
          myFunction(event, param1, param2);
        });

Learn more on MDN

Event Propagation and Bubbling

Events propagate through the DOM in two phases:

  • Capturing phase: The event travels down from the root to the target element
  • Bubbling phase: The event bubbles up from the target element to the root

Use event.stopPropagation() to prevgent further propagation of the current event in bubbling and capturing phases:


        element.addEventListener('click', function(event) {
          event.stopPropagation();
        });

Learn more on MDN

Working with Different Handler Functions and Scopes

Event handlers can be:

  • Inline event handlers (not recommended)
  • Named functions
  • Anonymous functions

Event handlers can have different scopes:

  • Global scope
  • Function scope
  • Lexical scope

Use bind() to change the value of this in event handlers:


        element.addEventListener('click', myFunction.bind(myObject));
